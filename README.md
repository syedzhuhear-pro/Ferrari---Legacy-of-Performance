## 📖 About

**Ferrari — Legacy of Performance** is a front-end web project dedicated to one of the most iconic automotive and motorsport brands in the world. The site walks through Ferrari's heritage — from Enzo Ferrari's founding vision to the brand's record-breaking achievements on the track — combining clean visuals with a red-and-black design language inspired by the Prancing Horse.

This project was built as a hands-on exercise in front-end web development, focusing on semantic HTML structure, responsive CSS layouts, and an engaging, brand-consistent user experience.

## ✨ Features

- 🏁 **Heritage & History** — A timeline-style overview of Ferrari's founding and evolution
- 🚗 **Iconic Models** — Showcase of legendary Ferrari vehicles across decades
- 🏆 **Racing Legacy** — Highlights of Ferrari's achievements in Formula 1 and motorsport
- 🎨 **Brand-Inspired Design** — Ferrari red (`#FF2800`) and black color palette with clean typography
- 📱 **Responsive Layout** — Adapts across desktop, tablet, and mobile screens
- ⚡ **Lightweight & Fast** — Pure HTML/CSS with no heavy frameworks, for quick load times

## 🚀 Getting Started

### Prerequisites
Just a modern web browser — no build tools or dependencies required.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/syedzhuhear-pro/Ferrari.git

# Navigate into the project folder
cd Ferrari

# Open index.html directly in your browser
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

Or simply open `index.html` with your file explorer / drag it into any browser tab.

### Deploy Your Own Copy
This project is already deployed via **GitHub Pages**. To deploy your own fork:
1. Go to your repository **Settings → Pages**
2. Select the branch (usually `main`) and root folder
3. Save — your site will be live at `https://<your-username>.github.io/Ferrari/`
